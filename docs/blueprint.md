# **App Name**: SyncStream

## Core Features:

- Secure Room Entry: User authentication and room management to ensure secure access and proper room allocation.
- Synchronized Playback: Synchronized video playback using React Player, ensuring all users in a room experience the video content in sync.
- Real-Time Chat: Real-time text chat via WebSocket for live interaction during video playback.

## Style Guidelines:

- Primary color: Dark blue (#1A237E) for a focused viewing experience.
- Secondary color: Light gray (#EEEEEE) for clean backgrounds and text contrast.
- Accent: Teal (#26A69A) for interactive elements and highlights.
- Clear and readable sans-serif fonts for UI text.
- Simple, outlined icons for controls.
- Clean, minimalist layout with clear divisions between video, chat, and controls.
